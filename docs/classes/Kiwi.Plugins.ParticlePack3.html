Kiwi.Plugins.ParticlePack3 Class

Defined in: src/_particlepack3.js:1
Module: Kiwi.Plugins
Parent Module: Kiwi

ParticlePack3 contains factory methods to create particles for your KiwiJS game. Use the create methods to generate particles or groups of particles.

For a walkthrough on setting up the particle pack plugin check the README.

startEmitting

(
  • loop
  • removeOnComplete
  • numParts
)

Defined in src/_particlepack3.js:128

Run startEmitting on all members of an effect group.

Parameters:

  • loop Boolean

    Set to true for continuous looping. Overrides and updates the config loop setting.

  • removeOnComplete Boolean

    If not looping, then the gameobject will destroy itself after one full emission cycle.

  • numParts Number

    Number of particles to generate, set on the config object - if not provided the current config value will be used

stopEmitting

(
  • immediate
  • remove
)

Defined in src/_particlepack3.js:151

Run stopEmitting on all members of an effect group.

Parameters:

  • immediate Boolean

    Stops the emitter and removes any existing particles.

  • remove Boolean

    If true the gameobject will mark itself for removal either immediately, or after a completed cycle.
